# Users

## User Object Definition

| Name       | Type   | Description        |
| ---------- | ------ | ------------------ |
| first_name | string | User first name    |
| last_name  | string | User last name     |
| email      | string | User email address |

> User Object

```shell
{
 "first_name":"John",
 "last_name":"Wick",
 "email":"john.wick@example.com"
}
```
